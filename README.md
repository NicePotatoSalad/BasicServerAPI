# Basic server API
### Frontend 
The most basic implementation of the button, made out of template

### Backend
The most basic implementation of the GET request to a localhost

### How to run
1. Start the backend:
    - cd backend-api
    - dotnet run
2. Start the frontend:
    - cd frontend-app
    - If npm not installed, run: npm install
    - npm run dev

### Tech stack 
- Frontend
    - React
    - Vite
- Backend
    - C# .NET
